## 📖 简介 (Introduction)

**AILinkX** 不是一个简单的聊天机器人，它是一个寄生于您 PC 中的**数字生命体**。

在信息爆炸的时代，我们被微信、钉钉等 IM 软件绑架，成为了信息的奴隶。AILinkX 的诞生旨在通过 AI Agent 技术，将您从繁杂的社交与事务中解放出来。

它拥有**独立的认知、记忆与执行能力**。它能读懂潜台词，能记住三个月前的承诺，甚至能自动帮您下载发票、预约会议。它运行在您的本地，绝对忠诚，绝对私密。

> **⚠️ 开发进度说明**：本项目目前正处于**封闭开发阶段**。核心功能架构已确定，正在进行稳定性测试与细节打磨。我们致力于交付一个稳定、安全且惊艳的产品，而非半成品。请耐心等待，我们很快就会与大家见面！

## 🔥 核心爆点 (Killer Features)

### 1. 🤖 智能事务官 (Agentic Workflow)
**别让聊天记录仅仅是聊天记录。**
AILinkX 具备自主执行权（Agency），能从碎片化的信息流中提取结构化任务：
- **自动归档**：识别群里的“发票”、“合同”文件，自动下载并按项目归档到本地文件夹。
- **日程预约**：识别“下周三下午两点开会”，自动写入系统日历。
- **财务助手**：OCR 识别转账/红包截图，自动生成 Excel 流水单。

### 2. 🧠 全息记忆宫殿 (Holographic Memory)
**打破“仅展示最近3天”的魔咒。**
- **本地 RAG 架构**：内置轻量级向量数据库，所有历史对话向量化存储。
- **语义检索**：支持问答式查询 —— “李总上个月说喜欢喝什么茶？”，AI 直接调取记忆回答，而非简单的关键词匹配。

### 3. 💓 情感雷达 (Social Intelligence)
**赋予你顶级的社交博弈能力。**
- **潜台词解码**：实时分析对方回复的微表情与情绪倾向（敷衍、愤怒、期待）。
- **矛盾猎手**：实时比对对方当前言论与历史记忆。如果对方说“在加班”，但历史画像显示他此时通常在打游戏，系统发出风险预警。

### 4. 🧬 数字分身养成 (Digital Twin)
**在赛博世界里，训练另一个“你”。**
- **风格克隆**：挂载您的历史语料，微调 LoRA 模型，让 AI 学会您的口头禅、语气助词和价值观。
- **自动托管**：在您休息时，授权 AI 分身接管常规社交，表现得天衣无缝。

## 🛡️ 安全与技术原理 (Safety & Tech)

我们深知账号安全是用户的底线。AILinkX 采用独创的 **"非侵入式 (Non-Intrusive)"** 架构：

* ✅ **不注入 DLL，不 Hook 内存**：完全不同于传统的协议挂或注入挂。
* ✅ **视觉感知**：像真人一样通过视觉识别 UI 元素，通过系统辅助接口模拟鼠标键盘操作。
* ✅ **拟人化引擎**：内置随机延迟、鼠标轨迹模拟算法，极大降低风控风险。
* ✅ **Local-First**：所有数据（记忆库、模型、配置）均存储于您本地电脑，**绝不上传云端**。
